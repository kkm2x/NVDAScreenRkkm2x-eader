# قارئ الشاشة NVDA للأندرويد

تطبيق قارئ شاشة متقدم للأندرويد يقرأ جميع محتويات الشاشة من أي تطبيق، مستوحى من NVDA الأصلي.

## الميزات الرئيسية

### 1. قراءة شاملة للشاشة
- قراءة جميع العناصر على الشاشة من أي تطبيق
- دعم كامل للغة العربية
- تحديث تلقائي عند تغيير محتوى الشاشة

### 2. التحكم الصوتي
- **سرعة القراءة**: قابلة للتعديل من 0.5x إلى 2.0x
- **مستوى الصوت**: قابل للتعديل من 0% إلى 100%
- **درجة الصوت**: قابلة للتعديل
- **نطق الرموز**: مستويات مختلفة من نطق الرموز

### 3. خدمة الوصولية
- تعمل على مستوى النظام
- تقرأ محتوى أي تطبيق مفتوح
- تتفاعل مع جميع أحداث الوصولية

### 4. الإعدادات المتقدمة
- حفظ واستعادة الإعدادات
- تخصيص كامل للصوت والسرعة
- دعم لغات متعددة

## المتطلبات

- Android 7.0 (API 24) أو أعلى
- 50 MB من مساحة التخزين
- إذن الوصولية (Accessibility Permission)

## التثبيت

### من ملف APK
1. قم بتحميل ملف APK من الإصدارات
2. افتح الملف على جهازك الأندرويد
3. اتبع خطوات التثبيت

### من المصدر
```bash
# استنساخ المشروع
git clone https://github.com/yourusername/NVDAScreenReader.git
cd NVDAScreenReader

# بناء التطبيق
./gradlew build

# تثبيت على جهاز متصل
./gradlew installDebug
```

## الملفات الرئيسية

### NVDAAccessibilityService.kt
خدمة الوصولية الرئيسية التي تقرأ جميع العناصر على الشاشة.

**المسؤوليات:**
- الاستماع إلى أحداث الوصولية
- قراءة محتوى الشاشة
- التحكم في Text-to-Speech

### MainActivity.kt
الشاشة الرئيسية للتطبيق.

**المسؤوليات:**
- عرض حالة الخدمة
- التحكم في السرعة والصوت
- فتح إعدادات الوصولية

## الإعدادات

### إعدادات الصوت
```kotlin
// السرعة (0.5 - 2.0)
textToSpeech.setSpeechRate(1.0f)

// درجة الصوت (0.0 - 2.0)
textToSpeech.setPitch(1.0f)

// مستوى الصوت (0.0 - 1.0)
// يتم التحكم فيه عبر منزلق الصوت
```

### إعدادات الوصولية
```xml
<!-- في accessibility_service_config.xml -->
android:accessibilityEventTypes="typeAllMask"
android:accessibilityFeedbackTypes="feedbackSpoken"
android:canPerformGestures="true"
```

## استكشاف الأخطاء

### الخدمة لا تعمل
1. تأكد من تفعيل الخدمة في إعدادات الوصولية
2. أعد تشغيل التطبيق
3. تحقق من أن التطبيق لديه إذن الوصولية

### لا يوجد صوت
1. تحقق من أن مستوى الصوت ليس على الصفر
2. تأكد من تفعيل الصوت على الجهاز
3. تحقق من أن محرك Text-to-Speech مثبت

### الخدمة بطيئة
1. قلل عدد التطبيقات المفتوحة
2. أعد تشغيل الجهاز
3. تحقق من مساحة التخزين المتاحة
